# WaveArt

Project for VandyHack2019 Nov 1-3

### Installation and use

Clone this repository:

```
git clone https://github.com/sircharlie/vandy2019.git
```

Go to __backend__ and run

```
npm install
```

```
npm start
```

Go to __frontend__ and run

```
yarn install
```
```
yarn watch
```
